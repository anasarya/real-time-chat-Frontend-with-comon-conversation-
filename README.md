# real-time-chat-Frontend-with-comon-conversation-
A modern, interactive chat application built with HTML, CSS, and JavaScript. Features include a scrollable chatbox, day/night mode toggle, neon banner, chatbot with common conversational replies, typing indicator, and persistent chat history using localStorage. Perfect for showcasing real-time front-end interactivity and animations.
Key highlights include:

Neon Banner: “Real-Time Chat App” with glowing animation.

Day/Night Mode Toggle: Switch between day and night themes, with the background and chatbox colors updating automatically.

Bot Typing Indicator: Simulates the bot “typing” for a realistic experience.

Persistent Chat History: All messages are saved in localStorage and persist across page refreshes.

Clear Chat Functionality: Users can clear the chatbox at any time.

Smooth Animations: Messages fade in, and the chat scrolls smoothly for an engaging UI.

Interactive Chatbot: Provides friendly responses for greetings, questions, small talk, and jokes.

This project demonstrates front-end development skills, including DOM manipulation, CSS animations, responsive UI design, localStorage usage, and basic chatbot logic. It’s perfect for showcasing real-time interactivity in a modern web interface.
